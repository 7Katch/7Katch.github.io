    Concetti chiave

    Appunti
    ✓
    Paradigma Cloud: Passaggio dai server on-premise ("Pets", curati singolarmente) al cloud ("Cattle", risorse usa e getta gestite in IaaS, PaaS, SaaS).
    ✓
    Servizi Core AWS: * *Storage e Database:* Object Storage (S3), Block Storage (EBS), File Storage (EFS), Database relazionali (RDS/Aurora) e NoSQL (DynamoDB, DocumentDB). * *Compute:* Macchine virtuali (EC2), Container orchestrati (EKS) e architetture Serverless (Lambda).
    ✓
    Sicurezza e Rete (VPC): Recinti virtuali, Subnet pubbliche/private, Security Group (Stateful), Network ACL (Stateless) e gestione degli accessi con IAM e KMS (Crittografia).
    ✓
    Infrastructure as Code (IaC): Gestione dichiarativa dell'infrastruttura tramite strumenti come **Terraform** e AWS CDK.
