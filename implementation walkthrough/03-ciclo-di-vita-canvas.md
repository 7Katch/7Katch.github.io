# Aggiornamento: Diagramma Canvas per Waterfall vs Agile

**Data**: 24 Giugno 2026
**File modificato**: `EDIDS/argomenti/02-ciclo-di-vita.html`

## Dettagli Modifica
I concetti chiave originariamente annotati nella sezione "Appunti" del file "02-ciclo-di-vita.html" sono stati estesi conformemente allo stile `SO`.
- È stata aggiunta un'`index-grid` per il supporto alla Sidebar dinamica e sostituito il tag script con `katchkit.js`.
- Nei blocchi descrittivi, è stato introdotto uno **script HTML5 Canvas** che disegna a tempo di esecuzione un diagramma visivo mettendo a confronto l'approccio lineare (Waterfall), illustrato con task in discesa a cascata, con l'approccio iterativo (Agile), illustrato tramite loop di Sprints concatenati che portano alla fase di "Release".
- Il design supporta estetica dark/glassmorphic. Nessun codice hardcoded al di fuori di etichette base relative alle sezioni e colori di design.
